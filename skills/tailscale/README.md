# Tailscale Skill

Manage Tailscale VPN connections, devices, and network configuration.

See [SKILL.md](SKILL.md) for full usage documentation.

## Quick Start

1. Install Tailscale: https://tailscale.com/download
2. Authenticate: `tailscale up`
3. Check status: `tailscale status`

## Features

- Connect/disconnect from Tailscale network
- List and manage devices
- Configure exit nodes
- Manage ACLs and sharing
- DNS configuration
